API Documentation

Access VotersSpeak data programmatically through our API.

Getting Started

The VotersSpeak API provides programmatic access to government official data, contact information, and more. To use the API, you'll need to register for an API key.

GET /api/v1/officials

Returns a list of all government officials.

Parameters

  • branch - Filter by government branch (house, senate, executive, etc.)
  • state - Filter by state (two-letter code)
  • party - Filter by political party
  • issue - Filter by issue area
GET /api/v1/officials/by-address

Returns officials representing a specific address.

Parameters

  • street - Street address
  • city - City
  • state - State (two-letter code)
  • zip - ZIP code
GET /api/v1/officials/{id}

Returns detailed information about a specific official.

POST /api/v1/contact

Send a message to an official through the API.

Parameters

  • official_id - ID of the official to contact
  • sender_name - Name of the sender
  • sender_email - Email of the sender
  • subject - Message subject
  • message - Message body
